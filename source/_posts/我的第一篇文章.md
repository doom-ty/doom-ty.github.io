### 二、安装所有文章模块

1、在博客根目录（注意不是yilia根目录）执行以下命令(确保node版本大于6.2)：
```
npm i hexo-generator-json-content --save
```
2、在根目录_config.yml里添加配置：
```
  jsonContent:
    meta: false
    pages: false
    posts:
      title: true
      date: true
      path: true
      text: false
      raw: false
      content: false
      slug: false
      updated: false
      comments: false
      link: false
      permalink: false
      excerpt: false
      categories: false
      tags: true
```
